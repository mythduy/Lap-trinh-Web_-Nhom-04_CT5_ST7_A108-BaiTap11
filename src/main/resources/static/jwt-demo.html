Hướng dẫn sử dụng:
  1. Đăng ký tài khoản mới hoặc sử dụng tài khoản có sẵn
  2. Đăng nhập để nhận JWT token
  3. Token sẽ được lưu tự động và hiển thị
  4. Test các API endpoint với token đã nhận
  5. Quan sát phân quyền theo role:
    • USER: Chỉ truy cập được /api/users/me
    • MANAGER: Truy cập được USER + /api/users + /api/manager/reports
    • ADMIN: Truy cập được tất cả
